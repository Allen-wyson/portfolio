Iterative Best Response of Zero-sum Racing Game

Applied game theory to a competitive two-player racing scenario modeled as a discrete zero-sum game. Implemented the Iterative Best Response (IBR) algorithm in Python to find Nash equilibria in racing strategy selection. Analyzed convergence behavior, strategy stability, and how changing the reward structure affects equilibrium outcomes.

Key Results

  • Implemented IBR algorithm converging to Nash equilibrium in ≤20 iterations
  • Validated zero-sum property: one player's gain equals the other's loss
  • Demonstrated sensitivity of equilibrium to reward matrix perturbations
  • Extended analysis to mixed-strategy equilibria using linear programming

Challenges & Solutions

The main challenge was ensuring the IBR algorithm converged rather than cycling between strategies. This was addressed by introducing a damping factor on strategy updates and checking for convergence using L2-norm thresholds. Mixed-strategy equilibria required switching to a linear program (scipy.optimize.linprog), which resolved non-convergence in non-strictly-determined games.
