EPFL · Sep 2024 – Dec 2024

Mobile Robotics with Thymio

Implemented navigation, localization, and obstacle avoidance for the Thymio II mobile robot using Python. Applied an Extended Kalman Filter (EKF) for state estimation, fusing odometry and proximity sensor data. Implemented global path planning (A*) and local obstacle avoidance, enabling the robot to navigate cluttered environments autonomously.

Key Results

  • EKF reduced localization error by ~60% vs. odometry-only estimate
  • A* path planning successfully navigated 5×5 grid environments
  • Local avoidance handled dynamic obstacle insertion in real-time
  • Full pipeline running at 10 Hz on-board Python controller

My Contribution

  • Implemented Extended Kalman Filter (EKF) for robot state estimation
  • Fused odometry and camera measurements to estimate robot position and orientation
  • EKF corrected cumulative errors (wheel slippage, sensor drift) to prevent trajectory deviation

Challenges & Solutions

The Thymio's proximity sensors have a narrow detection cone, causing missed detections at oblique angles. This was addressed by fusing multiple sensor readings with a confidence weighting scheme in the EKF update step. Odometry drift over long runs was corrected by periodic landmark-based re-localization using known map features.
